sudo fdisk /dev/xvdf
Press n to create new partition
Just press Enter (to accept the default p for primary).
Accept the default partition number (should be 1) → press Enter.
Accept the default first sector → press Enter.
Accept the default last sector (uses full disk) → press Enter.
Now type t to change the partition type.
Type 8e → press Enter (this sets it to Linux LVM).
Type p to verify — it should say Linux LVM.
Finally, type w to write the partition table.
